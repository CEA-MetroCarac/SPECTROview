# User Guide

Welcome to the SPECTROview User Guide. This guide covers everything you need to get started with spectroscopic data processing and visualization.

## Workspaces

SPECTROview is organized into three main workspaces:

| Workspace | Purpose |
|-----------|---------|
| [**Spectra**](spectra.md) | Process individual or multiple discrete spectra |
| [**Maps**](maps.md) | Process hyperspectral 2D maps and wafer data |
| [**Graphs**](graphs.md) | Visualize and plot data |

## Getting Started

1. **[Install SPECTROview](installation.md)** — pip install in one command
2. **[Check supported formats](supported-data.md)** — .wdf, .spc, .txt, .csv, .xlsx, .dat
3. **Open your data** — the app auto-routes to the correct workspace
4. **Process & visualize** — baseline, peaks, fitting, plotting

## Key Features

- **[Tensor Fit Engine](fitting.md)** — High-performance batched fitting (10–15× faster)
- **[Multivariate Analysis](mva.md)** — PCA and NMF for spectral decomposition
- **[Quick Calculators](calculators.md)** — Laser spot size, penetration depth, unit converter
- **[Keyboard shortcuts](shortcuts.md)** — Speed up your workflow
