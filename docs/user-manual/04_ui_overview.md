## 4. User Interface Overview

SPECTROview is meticulously designed for the efficient processing of spectroscopic data and the seamless visualization of fitted results. The interface is built around three primary workspaces, each tailored for a specific workflow:


- **Spectra**: Dedicated to processing one or multiple discrete spectra.
- **Maps**: Optimized for handling one or multiple hyperspectral datasets, including extensive wafer data and 2D maps.
- **Graphs**: Designed for robust plotting and comprehensive data visualization.

> **Tooltip**: Most GUI elements in SPECTROview (buttons, text boxes, dropdowns, etc.) include built-in tooltips. Simply hover your mouse cursor over an element for 1 second to view a brief explanation of its function.

| ![Clear](../user_manual_images/UI_Overview/imageFile9.png) | **Clear**: Clears the currently active workspace, removing all loaded data to start a fresh session. |

| ![Settings](../user_manual_images/UI_Overview/imageFile11.png) | ![Settings Panel](../user_manual_images/UI_Overview/imageFile14.png)<br> *Figure 2: Settings Panel: Opens the configuration panel where you can fine-tune mathematical fitting parameters and define the default storage directory for your custom fit models.*<br> |
| ![Theme](../user_manual_images/UI_Overview/imageFile15.png) | **Theme Toggle**: Instantly switches the application interface between Dark and Light modes. |

| ![About](../user_manual_images/UI_Overview/imageFile17.png) | **About**: Displays version information, licensing, and details about the SPECTROview application. |
