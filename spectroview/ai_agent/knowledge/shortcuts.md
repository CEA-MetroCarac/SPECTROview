# Purpose

This file contains the keyboard shortcuts available in SPECTROview. It is referenced when users ask how to access features quickly.

---

# Application Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + O` | Open file(s) |
| `Ctrl + S` | Save project |
| `Ctrl + Shift + S` | Open Settings dialog |
| `Ctrl + Shift + A` | Open / focus AI Chat Agent |
| `Ctrl + W` | Close current tab |
| `Ctrl + Q` | Quit application |
| `Ctrl + Z` | Undo (where applicable) |

---

# AI Chat Agent Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + Shift + A` | Open AI Chat Agent panel |
| `Enter` | Send the current message |
| `Shift + Enter` | Insert a new line in the input box |

---

# Graphs Workspace Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + P` | Create a new plot with current settings |
| `Ctrl + A` | Select all graphs in the MDI area |
| `Delete` | Delete the selected graph |
| `F5` | Refresh / re-render the current graph |
| `Ctrl + C` | Copy the current graph as image to clipboard |

---

# Spectra Workspace Shortcuts

| Shortcut | Action |
|----------|--------|
| `Space` | Fit the current spectrum |
| `Ctrl + F` | Run batch fit (VBF) |
| `Arrow Keys` | Navigate to previous / next spectrum |
| `Ctrl + D` | Toggle dark/light theme |
| `+` / `-` | Zoom in / out on the spectrum |

---

# Notes

- Shortcuts may vary slightly depending on the operating system (macOS uses `Cmd` in place of `Ctrl` for some actions).
- Shortcuts can be customised in **Settings → Keyboard** if available.
