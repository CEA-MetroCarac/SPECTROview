## User Interface Overview

SPECTROview is meticulously designed for the efficient processing of spectroscopic data and the seamless visualization of fitted results. The interface is built around three primary workspaces, each tailored for a specific workflow:


- **Spectra**: Dedicated to processing one or multiple discrete spectra.
- **Maps**: Optimized for handling one or multiple hyperspectral datasets, including extensive wafer data and 2D maps.
- **Graphs**: Designed for robust plotting and comprehensive data visualization.
